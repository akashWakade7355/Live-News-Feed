# ✨ Features

- 🔎 Search news using any keyword
- 🌆 Search news based on City
- 📰 Filter news by Category
- 📱 Fully Responsive Design
- 🎨 Beautiful Modern UI
- ⚡ Fast API Fetching using Async/Await
- 🖼️ News Cards with Images
- 📖 Read More button for full article
- 🌐 Live Headlines from NewsAPI
- ❌ Handles missing images gracefully
- 🔄 Dynamic Rendering using JavaScript
- 💻 Beginner Friendly Code Structure

# 🔑 Get Your NewsAPI Key

1. Visit

```
https://newsapi.org/
```

2. Create a free account.

3. Generate your API Key.

4. Replace it inside **script.js**

```javascript
const API_KEY = "YOUR_API_KEY";
```

      │
